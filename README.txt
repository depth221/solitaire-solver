# solitaire-solver
* 운영체제: Ubuntu 20.04 LTS
* 컴파일러: g++ (Ubuntu 9.4.0-1ubuntu1~20.04.2) 9.4.0
* make: GNU Make 4.2.1

## 컴파일 및 실행
```bash
make
./main (시드 값)
```

시드 값을 지정하지 않으면 0으로 설정됩니다.

## 출력
### 초기 세팅

(stock) (waste)        (output-스페이드, 하트, 클로버, 다이아몬드)

(playing #1) (playing #2) ... (playing #7)
             (playing #2) ... (playing #7)
                          ... (playing #7)
                              (playing #7)

이때 파란색 글자는 카드가 뒤집어져 있다는 것을 의미합니다.

예시)
◆ Q                                
                                   
♠ Q  ◆ A  ♥ 2  ♣ 3  ♣ K  ♥ 3  ♠ 4  
      ♠ A  ♣ A  ♠ K  ♥ K  ♣ J  ◆ 2  
            ♠ 3  ◆ K  ♠ 5  ◆ 3  ♥ 5  
                  ♥ 6  ♠ 6  ♣ 7  ♣ 2  
                        ◆ J  ♠ 8  ♥ 7  
                              ♣ 6  ◆ 9  
                                    ♥ J
